Glossary
=========

Verbs
-----

.. glossary::
	
	ir
		To go.

		* voy, vas, va, vamos, vais, van
		* fui, fuiste, fue, fuimos, fuisteis, fueron

	hacer
		To do / to make

		* hago, haces, hace, hacemos, hacéis, hacen
		* hice, hiciste, hizo, hicimos, hicisteis, hicieron

	estar
		To be (or to have been)

		Use this to say that you were somewhere.

		* estoy, estás, está, estamos, estáis, están
		* estuve, estuviste, estuvo, estuvimos, estuvisteis, estuvieron

		.. admonition:: Examples

			* ¿Dónde estuviste el lunes?

				* Where were you on Monday?
		
	aprender
		To learn

		* aprendo, aprendes, aprende, aprendemos, aprendéis, aprenden
		* aprendí, aprendiste, aprendió, aprendimos, aprendisteis, aprendieron

	ser
	 	To be (permanent) or to have gone.

		Use this to say that you went somewhere.

		* soy, eres, es, somos, sois, son
		* fui, fuiste, fue, fuimos, fuisteis, fueron

		.. admonition:: Examples

			* Fui a España hace mucho tiempo

				* I went to Spain a long time ago.

	escribir
		To write

		* escribo, escribas, escriba
